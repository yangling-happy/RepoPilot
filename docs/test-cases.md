# RepoPilot 功能测试用例

## 1. 文档生成模块

### 增量刷新 (refresh)
- HEAD一致跳过 → 返回 "No new commits"，不生成文档
- 正常增量生成 → 生成变更文件的文档，返回新commit列表
- 无变更文件 → 创建任务但不生成文档文件
- 删除文件处理 → 标记文件为 "File deleted"，docFilePath为空
- 不支持的文件类型 → 跳过该文件，不调用GitLab API
- 仓库不存在 → 抛出异常，提示仓库不存在
- 多个commit增量 → 为所有变更文件生成文档

### 全量扫描 (scanLocal)
- 正常扫描 → 生成所有Java文件的文档
- .gitignore过滤 → 被忽略的文件不生成文档
- 无Java文件 → scannedFileCount > 0，generatedFileCount = 0
- 多类型单文件 → 文档中包含所有类型的文档
- HEAD解析失败 → 抛出异常，提示无法解析HEAD

### 重建 (rebuild)
- 正常重建 → 通过GitLab API获取文件并生成文档
- commit不存在 → 抛出异常

### 查询 (query)
- 按项目查询 → 返回该项目所有文档
- 按分支过滤 → 只返回指定分支的文档
- 按文件路径过滤 → 只返回匹配的文件文档
- 按commit查询 → 只返回该commit的文档
- 去重逻辑 → 每个文件只返回最新版本
- 无结果 → 返回空列表

### JavaDoc 解析
- 类解析 → 正确提取参数、返回值、异常信息
- 接口解析 → kind = "INTERFACE"
- 枚举解析 → kind = "ENUM"，包含枚举值
- 注解解析 → kind = "ANNOTATION"
- 泛型参数 → 正确分割泛型参数

## 2. 部署流水线模块

### 触发部署 (trigger)
- 正常触发 → 创建部署任务，返回任务ID
- 参数校验-缺少project → 返回 400 错误
- 参数校验-缺少branch → 返回 400 错误
- 参数校验-缺少token → 返回 400 错误
- 并发触发 → 应提示任务冲突或排队

### 构建任务 (build)
- 正常构建 → 构建成功，输出日志
- 构建失败 → 构建失败，返回错误信息
- 构建取消 → 构建终止，状态更新

### 部署任务 (deploy)
- 正常部署 → 部署成功
- 自定义deploy.sh → 执行自定义脚本
- SSH密钥配置 → 自动配置SSH密钥
- 部署失败 → 返回失败信息

### 任务管理
- 查询任务状态 → 返回任务当前状态
- 取消任务 → 任务终止
- 查看日志 → 实时显示构建/部署日志

---

## 3. 终端服务模块

### WebSocket 连接
- 正常连接 → 连接成功
- 连接断开重连 → 重新建立连接
- 认证失败 → 连接被拒绝

### 任务执行
- 异步任务执行 → 任务异步执行，日志实时推送
- 同步任务执行 → 等待任务完成，返回结果
- 超时终止 → 任务被终止
- 手动取消 → 任务终止

### 日志推送
- stdout推送 → stdout内容实时推送到前端
- stderr推送 → stderr内容实时推送到前端
- 结果解析 → 正确解析结构化结果

---

## 4. GitLab 集成模块

### OAuth 认证
- 正常登录 → 登录成功，获取用户信息
- 授权拒绝 → 返回错误提示
- Token过期 → 提示重新登录

### 仓库操作
- 克隆仓库 → 仓库克隆成功
- 克隆失败-无权限 → 返回权限错误
- 读取文件内容 → 返回文件内容
- 获取commit列表 → 返回commit列表

---

## 5. 前端模块

### 文档查看页
- 文档列表展示 → 显示文档列表
- 文档详情查看 → 显示结构化文档内容
- 空状态 → 显示空状态提示

### 产品文档页
- 刷新文档 → 触发增量刷新
- 全量扫描 → 触发全量扫描
- 进度展示 → 实时显示任务进度

### 产品部署页
- 触发部署 → 创建部署任务
- 终端日志 → 实时显示构建/部署日志
- 取消部署 → 任务终止

### 登录页
- GitLab登录 → 跳转GitLab授权页
- 登录回调 → 登录成功，跳转首页
- 未登录访问 → 跳转登录页
